# Intro

现代音乐也被称为“二十世纪音乐“

* 19世纪末发起
* 变化之大超过了以往所有音乐变化的总和， 这和二十世纪自然科学的变化是一致的
* 因此，很难用一种风格来界定
  * 反叛
  * 标新
  * 多元化
